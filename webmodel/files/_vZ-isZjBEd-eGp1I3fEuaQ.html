Title
Description "This rule looks for types that override Object.Equals(object) but do not provide an Equals(x) overload using the type. Such an overload removes the need to cast the object to the correct type. For value types this also removes the costly boxing operations. Assemblies targeting .NET 2.0 (and later) should also implement System.IEquatable." [1]
