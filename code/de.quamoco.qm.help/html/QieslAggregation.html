QIESL Aggregation

An aggregation where the specification is defined in QIESL.

General Rules

  1. The final result of the aggregation is expected to be assigned to the pre-defined variable result.

Attributes

Specification

  1. This contains the actual QIESL specification for the aggregation.
  2. The box below it shows either all ok or errors in the QIESL specification.

For all other attributes and general hints, see the description of aggregation.
