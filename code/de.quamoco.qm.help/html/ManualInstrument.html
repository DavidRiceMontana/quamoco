Manual Instrument

A manual instrument describes a manual way to collect the data for a measure. This can, for example, be part of a code review.

Attributes

Name

  1. You can change the default name Manual Instrument.
  2. If there is no specific reason to do otherwise, only capitalise the first word in the name.

Title

  1. A title can be an alternative, more readable identifier for the tool.
  2. Providing a title is optional
  3. If there is no specific reason to do otherwise, only captialise the first word in the title.

Description

  1. Describe in detail how a person that is responsible for the manual data collection can get the necessary data.
  2. Describe on what type of scale the result is expected (e.g. findings, points, etc.).

For all other attributes and general hints, see the description of instrument.
