Text Evaluation

An evaluation where the specification is a semi-formal text.

General Rules

  1. Within one model, the textual specification should be used consistently and marked out by a telling name.

Attributes

Specification

  1. The textual specification can be used for writing any syntax; thus the text will not be evaluated.

For all other attributes and general hints, see the description of evaluation.
