QIESL

Quamoco Impact Evaluation Specification Language (QIESL) defines a simple semi-functional language. As interpreter, Apache-Jaxl is used. For an exact definition of it see here:

Detailed Guide to QIESL Syntax

QIESL supports the following functions:
  • intersect, union on Findings
  • max,min,median,mean on Points 
  • linearDistribution(proportion,
    lowerBound,
    lowerResult,
    upperBound,
    upperResult)
    realizes a linear distribution, as shown in the following figure
  • Shortcuts of linearDistribution: 
    • linearDistribution(maxPoints, proportion) = linearDistribution(proportion, 0, 0, 1, maxPoints)
    • negativeLinearDistribution(maxPoints, proportion) = linearDistribution(proportion, 0, maxPoints, 1, 0)
  • methodRange, classRange, fileRange on Findings.
  • extent, calculates the size of finding-lists and of ranges.

QIESL is used for the QIESL Evaluation and the QIESL Aggregation.
