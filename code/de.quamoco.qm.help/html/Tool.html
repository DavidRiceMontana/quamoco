Tool

A tool is a software program, which is used to collect measurement data. This can be, for example, a static analysis tool.

General Rules

  1. Describe the used tool as precisely as possible.
  2. You can create different tools for different tool versions, if necessary.

Attributes

Qualified Name

  1. The qualified name is generated automatically based on the name.

Name

  1. You should choose a short descriptive name for the tool. Java.
  2. If there is no specific reason to do otherwise, only captialise the first word in the name.

Title

  1. A title can be an alternative, more readable identifier for the tool.
  2. This is optional.
  3. If there is no specific reason to do otherwise, only captialise the first word in the title.

Description

  1. Give more details on the tools.
  2. If applicable, give a URL where the tools can be found.

Invoked by

  1. Add the instruments that invoke this tool.
