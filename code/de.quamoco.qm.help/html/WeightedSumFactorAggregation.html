Weighted Sum Factor Aggregation

An evaluation by calculating the weighted sum from the evaluation results of various factors.

General Rules

  1. Use this evaluation when evaluating a factor that is refined by further factors.

Attributes

Aggregation Table

  1. Define Ranking or CP (Contribution Points) for all refining factors.
  2. If rank or points are set to zero the factor is ignored.

For all other attributes and general hints, see the description of evaluation.
