Single Measure Evaluation

An evaluation by applying pre-defined operations on a single measure.

General Rules

  1. Use this evaluation for factors that are measured by a single measure.

Attributes

Measure used for Evaluation

  1. There should be typically only one measure available.
  2. If the factor is measured by more than one measure, select the measure that shall be used for this evaluation.

Measure used for Normalization

  1. Define a normalization measure for normalizing the measure that is used for evaluation.

Range

  1. Normalize a measure of type finding by the suitable entity.
  2. Available ranges are methodRange, classRange, and fileRange.

Function

  1. Select the function (Linear Increasing Points or Linear Decreasing Points) and define upper and lower bounds.

For all other attributes and general hints, see the description of evaluation.
