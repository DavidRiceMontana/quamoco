Tag

A tag is a keyword that can be added to model elements.

General Rules

  1. Tags can be assigned to various model elements (e.g. Entities, Factors, or Measures).

Attributes

Qualified Name

  1. The qualified name is generated automatically based on the name.

Name

  1. You should choose a short descriptive name for the tag.
  2. If there is no specific reason to do otherwise, only captialise the first word in the name.

Title

  1. A title can be an alternative, more readable identifier for the tag.
  2. This is optional.
  3. If there is no specific reason to do otherwise, only captialise the first word in the title.

Description

  1. Give more details on the tag.

Used by

  1. Assign the tag to an element.
