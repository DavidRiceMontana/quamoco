Aggregation

An aggregation describes how to calculate the result of a measure from the results of other measures. There are various ways to calculate an aggregation, i.e., by using the union of findings or the mean of numbers (depending on the measure type), as well as writting QIESL syntax or a simple textual specification.

General Rules

  1. Only use measures that refine the current measure.

Attributes

Qualified Name

  1. The qualified name is generated automatically.

Name

  1. The default name is Aggregation. You can optionally give it a specific name, for example, if you want to have different evaluations for a factor.

Title

  1. A title is an optional additional name for the aggregation.

Description

  1. Optional element that can be used to textually describe and explain the aggregation.
