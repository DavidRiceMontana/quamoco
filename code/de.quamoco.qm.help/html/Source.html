Source

A source is a reference document that is the basis for the model element it is associated with. For example, the quality attributes from ISO 25010 are associated with the this source description.

General Rules

  1. Always when you take existing text or knowledge from a reference, cite it with the corresponding source.
  2. Source can be referenced by various model elements (e.g. Entities, Factors, or Measures).

Attributes

Qualified Name

  1. The qualified name is the identifier used to refer to the source. It is generated automatically by the name.

Name

  1. Give a short expressive name of the source.

Title

  1. If useful, add a title as an optional additional name for the source.

Description

  1. This should contain more information about the source, for example, where it is accessible on the Internet.

Referenced by

  1. Let an element reference to this source.
