Annotations

All model elements can be labelled with any number of annotations. An annotation is a pair of a key with a corresponding value.

General Rules

  1. Use annotations only if there is no suitable attribute available for this purpose.

Attributes

Key/Value Table

  1. The key has to be unique for the current model element.
  2. The value can be any text describing the facts for the particular key.
