Metric Based Instrument

A metric based instrument is a concrete description how to collect the data for a measure using a tool that calculates a numerical metric. Examples are lines of code or clone coverage.

Attributes

Tool

  1. Set the tool, which can calculate the metric.

Metric

  1. Use exactly the name and capitalizing of the metric as implemented in the tool.

For all other attributes and general hints, see the description of instrument.
